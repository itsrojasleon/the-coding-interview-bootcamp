<p className="instructions">Given a string, return true if the string is a palindrome or false if it is not.</p>
<p className="instructions">Palindromes are strings that form the same word if it is reversed.</p>
<p className="instructions">Do include spaces and punctuation in determining if the string is a palindrome.</p>

> <p className="ways">Two differents ways</p>

```
// first (ðŸ’)
function palindrome(str) {
  const reversed = str
    .split('')
    .reverse()
    .join('');
  return reversed === str;
  // should return true or false
}

// Second (ðŸ‘)
function palindrome(str) {
  return str.split('').every((char, i) => {
    return char === str[str.length - i - 1];
  });
}
```
