# Reverse a String

_Instructions:_

- Given a string, return a new string with the reversed order of characters.

_Examples:_

- `reverse('apple') === 'leppa'`

- `reverse('hello') === 'olleh'`

- `reverse('Greetings!') === '!sgniteerG'`

_Code:_

```javascript
// First (ðŸ)
function reverseString(str) {
  return str
    .split('')
    .reverse()
    .join();
}

// Second (ðŸŽ)
function reverseString(str) {
  let reversed = '';

  for (let char of str) {
    reversed = char + reversed;
  }
  return reversed;
}

// Third (ðŸ)
function reverseString(str) {
  return str.split('').reduce((rev, char) => char + rev, '');
}
```
